# EasyHardware

## Descripción del Proyecto

EASYHARDWARE es una aplicación web diseñada para proporcionar una solución integral en el sector de la ferretería. La plataforma facilita la **gestión del inventario**, el manejo de **usuarios y roles**, además de ofrecer herramientas para la generación de reportes, tales como **PDFs**, **Excel** y gráficos del negocio. Con una interfaz dinámica que integra jQuery y el robusto entorno de ASP.NET Core 6, el proyecto se convierte en una herramienta multifuncional adaptable tanto a las necesidades del usuario final como a los requerimientos operacionales de una ferretería moderna.

## Características

- **Manipulación de Inventario:** Sistema para el seguimiento y actualización de stock.
- **Gestión de Usuarios y Roles:** Administración de accesos y permisos para usuarios.
- **Manipulacion de correos:** Implementación de modulos de SMTP para envio de correos empresariales.
- **Generación de Reportes:** Exportación de datos en formato PDF y Excel, junto con visualizaciones gráficas para el análisis del negocio.
- **Interactividad Dinámica:** Implementación de jQuery para una experiencia de usuario fluida y responsiva.

## Tecnologías Utilizadas

- **Frontend:** jQuery para interacciones dinámicas, BoostTrap.
- **Backend:** ASP.NET Core 6 con C#, Entity Framework.
- **Mapeo de Objetos:** Automapper para la transformación eficiente de datos.
- **Servicios en la Nube:** Firebase para almacenamiento y servicios complementarios.
- **Herramientas Adicionales:** Integración de bibliotecas para la generación y manejo de archivos PDF, Excel y gráficos.
